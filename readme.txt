## this repository is for the retrevial, processing, and creating figures
## for the masters thesis of Jazmine B Russel, 2018 while at Arizona State University

## if you wish to use the landsat-data-extract scirpt to extract landsat data yourself
## you need to download the analysis ready landsat data for the dates you desire
## unzip them such that in each of the satellite subfolders
## each subfolder represents 1 day of observations
## also update the file my_sites.csv to locations you wish to extract data at

## example:
## data/landsat/Landsat7/<your_landsat7_data_with_desired_date_and_location_here>
## data/landsat/Landsat8/<your_landsat8_data_with_desired_date_and_location_here>
